# User Stories

## Visitor

- As a visitor, I want to browse available trips so that I can understand what the platform offers before registering.
- As a visitor, I want to search trips by title, description, or location so that I can find trips relevant to my interests.
- As a visitor, I want to view trip details so that I can check date, duration, price, capacity, and status.
- As a visitor, I want to explore trips grouped by location so that I can compare destinations.

## Registered User

- As a user, I want to create an account so that I can book trips.
- As a user, I want to log in securely so that I can access my bookings.
- As a user, I want to select travelers for a trip so that my booking reflects the correct number of seats.
- As a user, I want to complete a simulated payment so that my booking can be confirmed in the academic demo.
- As a user, I want to view my own bookings so that I can track pending, confirmed, and cancelled reservations.
- As a user, I want to cancel a pending booking so that I can remove a reservation before payment.

## Admin

- As an admin, I want to create trips so that the catalog can be managed from the web application.
- As an admin, I want to edit trips so that dates, prices, capacity, status, descriptions, images, and locations stay current.
- As an admin, I want to view users so that I can review registered accounts.
- As an admin, I want to view all bookings so that I can monitor platform activity.
